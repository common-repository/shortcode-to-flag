== Description ==

Shortcode to flag is an easy plugin, that will allow you to insert flags with shortcodes. For example, US will change to an American flag that wonâ€™t break your formatting. The height of the flags is 40pixels.

== Usage == 

Put [country code](http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) (in capital letters) to shortcode to display desired flag. 

e.g. [US]

== Edits == 

If you want to change height of the flags, do so in file flagstyle.css

== Installation ==

1. Upload `shortcode-to-flag` directory and it's contents to to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
